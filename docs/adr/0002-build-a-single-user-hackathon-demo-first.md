# Build a single-user hackathon demo first

Clue will target a polished single-user hackathon demo before becoming a productized multi-user app. The first build excludes auth, teams, sharing, permissions, and collaboration so the project can focus on the judged experience: add pins, have Cognee remember them, draw defensible strings, and explain those strings clearly. This keeps the reveal strong and avoids spending hackathon time on surfaces that do not prove the core idea.
